How To Play

1️⃣ Downloading

First of all you have to load the main game (~36GB). Just use one of these links:

Full Client April 2025:
Download Full Client

Multipart Download April 2025:
Download Multipart Version

If you get a "download quota exceeded" error:

  1. Add shortcut to your Google Drive
  2. Create a new folder
  3. Move shortcut into that folder
  4. Right click the folder and download it

2️⃣ Unzipping

After downloading, unzip the files using WinRAR or 7Zip. Windows default unpacking does NOT work.

For Multipart: Put all files into one folder and extract the first part. The rest extracts automatically.

Make sure files end with:

  • .part01
  • .part02

NOT with -001 or -002. Delete those numbers if needed.

OneDrive does NOT count as a real drive and will not work as install location.

3️⃣ Starting the Launcher

Run ddo_launcher.exe as Administrator.

If prompted to install .NET 9.0, it's safe to install.

Make sure DirectX is installed to avoid d3dx9 and XINPUT errors.

4️⃣ Configuring the Launcher

Check that server settings match:

Server: ddon.org
Lobby IP: ddon.org — Port 52100
DL IP: ddon.org — Port 52099

5️⃣ Registering an Account

Register inside the launcher using Register button.

Choose any account ID and password. Write them down.

6️⃣ Starting the Game & Update Translation

You can click Login and start playing.

Or press the small translation icon inside launcher to install English patch.

Keep in mind some UI parts remain Japanese.

7️⃣ Beginning the Game

After character creation you enter the tutorial.

Complete the Green Tutorial quests to unlock features.

Follow red quest markers for main story.

Use borderless window mode for best experience.

If you need help, visit tech-support channel.

Have fun and enjoy Lestania, Arisen!
